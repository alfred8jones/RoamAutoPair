# Roam Disable Auto Pair

 
Now you can disable the auto completed pairs of double brackets and parentheses!



In action:
![Example](https://raw.githubusercontent.com/alfred8jones/RoamAutoPair/main/example.gif)



There's no more to it, no additional requirements needed, simply use the toggle switch and you're done!
